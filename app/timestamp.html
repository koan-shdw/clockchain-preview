GETTime API

Latest consensus block time and height.

curl "https://node.clockchain.network/api/time/time" \
  -H "Accept: application/json" \
  -H "x-api-key: YOUR_API_KEY"
{
  "success": true,
  "data": {
    "latestBlockTime": "—Z",
    "latestBlockHeight": "—"
  }
}
GETTimestamp API

A signed, anchored timestamp in UTC, GPS, TAI, ISO 8601, and RFC 3339.

curl "https://node.clockchain.network/api/time/timestamp" \
  -H "Accept: application/json" \
  -H "x-api-key: YOUR_API_KEY"
GETBlock Height API

Look up any historical block by height and verify its timestamp.

curl "https://node.clockchain.network/api/time/block?height=36042" \
  -H "Accept: application/json" \
  -H "x-api-key: YOUR_API_KEY"
